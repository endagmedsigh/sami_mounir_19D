Element

Element är själva linen av kod man skriver i HTML, den har en opening tag och en closing tag. I tagen finns det innehåll av olika liner av koder.

HTML

HTML står för HyperText Markup Language och det är standar språket som används för Hemsidor i World Wide Web, HTML överförs över internet med HTTP.

ID

ID är en unik id för en element. När man ger en ID kan man länka till olika delar av sidan som jag har gjort, om du trycker på navbar kan du länka till olika delar av sidan.

Class

Class fungerar nästan som ID men skillnaden är att Class kan användas till flera element på sidan, class kan inte direktlänka till delar av en sida jämför med ID.

Meta

Meta är hur skärment ska visa hemsidan eller reloda sidan

Responsive

Responsive är att skapa en hemsida som passar på alla enheter

HTTP

HTTP står för Hypertext Transfer Protocol som används för att överföra hemsidor till WWW (Worl Wide Web)

HTTPS

HTTPS står för Hypertext Transfer Protocol som används för att överföra hemsidor till WWW (Worl Wide Web) fast krypterad, så det är säkrare.

Compressed Data

HTTPS står för Hypertext Transfer Protocol som används för att överföra hemsidor till WWW (Worl Wide Web) fast krypterad, så det är säkrare.

CDN

CDN står för Content Delivery Network eller på svenska innehållsleveransnätverk är för att tillhandhålla nattjänster nära användare för ökad driftsäkerhet.

Local bootstrap

Är när ens bootstrap är på en local hårddisk/server och inte hämtad driekt från bootsrap som då är alltid uppdaterad

JQuery

Är en javascript biboltek gjord för att förenkla HTML och CSS som tex animeringa och AJAX för att snabba på webbutvecklingen, den släpptes 2006 av John Resig och 35% av dom mest besökta hemsidor använder sig av denna biblotek vilket gör denna till den populäraste javascript-bibloteket. Företag som använder jQuery är Google, Microsoft, IBM och Netflix.
